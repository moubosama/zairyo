- **合格ライン（差1枚以内 or ±10%）**: 正解はすべて集計表の発注数量列÷9戸（係数はX列の実値）

  | 名称 | 正解/戸 | 集計表の根拠 |
  |---|---|---|
  | 壁 石膏ボード | 87.2枚 | 122.061㎡÷**1.4**（AB列） |
  | 壁 耐水石膏ボード | 4.6枚 | 6.454㎡÷1.4（AC列） |
  | 天井 石膏ボード | 41.5枚 | 59.087㎡÷**1.45**+ﾊﾟｳﾀﾞｰ調整7枚（AD列） |
  | 下り天井 石膏ボード | 3.6枚 | 5.233㎡÷1.45（AE列） |
  | 一部界壁 石膏ボード | 3.4枚 | 5.047㎡÷**1.5**（Z列） |
